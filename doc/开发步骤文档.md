# 博客系统开发步骤

## 阶段 1：项目骨架与基础设施
### 后端
- 使用 Spring Initializr 创建 Spring Boot 3 项目，包含 Web、Security、JPA、Redis、Validation、Flyway 等依赖。
- 规划分层与包结构（auth/content/interaction/analytics/common）。
- 配置 `application-*.yml`、日志体系、全局异常与统一响应格式。

### 前端
- 使用 Vite 创建 Vue 3 + TypeScript + Pinia + Vue Router 项目。
- 规划路由结构与基础布局（首页、文章详情、登录、后台框架）。

## 阶段 2：核心业务（MVP）
### 数据库与迁移
- 根据设计文档编写 Flyway 脚本，创建 `users`、`roles`、`posts`、`tags`、`comments` 等核心表结构与索引。

### 后端
- 实现注册、登录、鉴权流程（JWT 双 Token + Redis 刷新 Token）。
- 完成文章增删改查、标签管理、基础评论（可选审核）。

### 前端
- 完成首页文章列表、文章详情、登录注册、后台文章管理页面。
- 封装 Axios、实现鉴权拦截器、编写 `useAuthStore`、`usePostStore`。

## 阶段 3：增强功能与缓存/统计
- 点赞、收藏、订阅、PV/UV 统计接口及后台仪表盘。
- 评论嵌套结构、审核流转逻辑。
- 接入 Redis 缓存策略与限流（登录、评论等），完善 Dashboard 数据展示。

## 阶段 4：搜索、优化与安全细节
- 按文档方案实现全文搜索（可选 MySQL ngram 全文索引或专门搜索组件）。
- 补齐安全防护：XSS 过滤、敏感词筛查、操作审计、权限校验。
- 性能与稳定：索引调优、缓存策略复查、定时任务刷量与清理。

## 阶段 5：部署与运维
- 按设计提供 `docker-compose.yml`、前后端 Dockerfile、`nginx.conf`，并完成联调。
- 配置 Prometheus + Grafana 采集 Spring Boot Actuator 指标。
- 视需求补充 GitHub Actions / CI-CD 流程，实现一键构建发布。
